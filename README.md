Kullanılan Teknolojiler:

* .Net Core Framework MVC
* MSSQL
* Generic Repository, EntityFrameWork ve Code first yaklaşımı
* Çok Katmanlı Yazılım Mimarisi
* Dependency Injection
* Identity FrameWork Core
* Partial View
* Boostrap
